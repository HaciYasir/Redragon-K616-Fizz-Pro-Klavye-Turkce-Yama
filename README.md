# Redragon-K616-RGB-Turkce-Yama
Redragon K616-RGB klavye sürücüsü için %100 Türkçe dil yaması.

Kurulum Adımları

1. Repositoride bulunan `tr` klasörünü indirin.
2. Klavyenizin yazılımının kurulu olduğu ana klasöre gidin (Örn: `C:\Program Files (x86)\Redragon...`).
3. İndirdiğiniz `tr` klasörünü `Text` klasörünün içine yapıştırın.
4. Ana klasörde bulunan `config.ini` ya da `Cfg` dosyasını Not Defteri ile açın.
